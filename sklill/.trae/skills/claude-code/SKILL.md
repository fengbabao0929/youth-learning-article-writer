# Claude Code Skill

This skill provides integration with Claude AI for various code-related tasks, including:

## Features

- **Code Generation**: Generate code snippets in multiple programming languages
- **Code Explanation**: Explain complex code logic and algorithms
- **Bug Fixing**: Identify and fix bugs in existing code
- **Code Optimization**: Improve code performance and readability
- **Documentation**: Generate documentation for codebase

## Usage Guidelines

### When to Invoke

Invoke this skill when:
- User asks for code in any programming language
- User needs help understanding existing code
- User wants to debug code issues
- User requests code optimization
- User needs documentation for their code

### Example Prompts

1. "Generate a Python function to calculate Fibonacci numbers"
2. "Explain how this JavaScript async function works"
3. "Find and fix the bug in this Java code"
4. "Optimize this SQL query for better performance"
5. "Create documentation for this React component"

## Supported Languages

- Python
- JavaScript/TypeScript
- Java
- C/C++
- C#
- Go
- Rust
- Ruby
- PHP
- SQL
- HTML/CSS
- And many more

## Integration Details

This skill leverages Claude AI's advanced coding capabilities to provide comprehensive code assistance. It can handle both simple code snippets and complex multi-file projects.

## Best Practices

- Provide clear and specific requirements for code generation
- Include relevant context when asking for code explanation
- Share complete code when requesting bug fixes
- Specify performance goals when asking for optimization
- Mention documentation format preferences when needed
