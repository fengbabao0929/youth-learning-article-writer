# 青少年学习力文章写作助手

## 描述
专门用于撰写青少年学习力提升类公众号文章的 AI 助手。基于爆款文章分析，生成高质量的教育类内容。

## 作者
学习教练八宝

## 版本
1.0.0

---

## Skill 功能

### 1. 选题生成
根据选题库或用户需求，生成适合公众号的文章选题。

**触发方式：**
- "帮我生成3个选题"
- "关于学习动力的文章选题"
- "这周要写什么选题"

**输出格式：**
- 选题标题
- 核心痛点
- 解决方向
- 预计字数

---

### 2. 标题生成
根据文章内容，生成 10-20 个爆款标题候选，覆盖不同标题套路。

**触发方式：**
- "帮我生成标题"
- "给这篇文章起个标题"
- "优化这个标题：xxx"

**标题类型：**
- 数字+痛点
- 疑问式
- 对立冲突
- 情绪激发
- 负面警示
- 颠覆认知
- 场景代入
- 方法承诺
- 结果对比
- 真实案例

---

### 3. 大纲生成
用 SCQA 框架生成文章大纲结构。

**触发方式：**
- "帮我写个大纲"
- "生成文章大纲：xxx"
- "规划文章结构"

**SCQA 框架：**
- **S (情境)**：描述家长熟悉的场景/问题
- **C (冲突)**：点出问题的严重性/常见误区
- **Q (问题)**：提出关键问题
- **A (答案)**：给出解决方案

**输出结构：**
```
## 一、开头（情境引入）
- 场景描述
- 痛点共鸣

## 二、问题分析（冲突+提问）
- 常见误区
- 问题根源
- 关键提问

## 三、解决方案（核心内容）
- 方法1：xxx
  - 具体步骤
  - 操作要点
- 方法2：xxx
  - 具体步骤
  - 操作要点
- 方法3：xxx
  - 具体步骤
  - 操作要点

## 四、案例说明（可选）
- 真实案例描述
- 问题解决过程
- 效果展示

## 五、结尾（行动建议）
- 总结要点
- 行动建议
- 引导关注/咨询
```

---

### 4. 正文写作
根据大纲写出完整的公众号文章。

**触发方式：**
- "帮我写这篇文章"
- "根据大纲写正文"
- "写一篇关于xxx的文章"

**写作要求：**
- **字数**：2000-3000字
- **风格**：专业但不严肃，有叙事感和真人温度
- **结构**：SCQA 框架
- **开头**：场景化描述，引发共鸣
- **中段**：分析+方法（配图建议）
- **结尾**：行动建议+引导关注
- **语言**：口语化，避免学术词汇
- **排版**：适合手机阅读（短段落+小标题）

**⭐ 重点要求：叙事感与真人温度**
- **完整句型**：使用完整的句子表达，避免碎片化短语
- **段落连贯**：段落之间有逻辑衔接，不是孤立的观点堆砌
- **叙事流畅**：像讲故事一样娓娓道来，有起承转合
- **情感温度**：用"我"的视角分享，带有个人的观察和感受
- **细节描写**：加入具体的场景、对话、细节，让内容更生动
- **避免列表体**：不要过度使用1.2.3.4的列表形式，用段落自然展开

### 6. 金句提炼
从文章中提炼适合传播的金句/口号。**重点是反认知金句**。

**触发方式：**
- "提炼金句"
- "生成传播口号"
- "总结一句话"
- "提炼反认知金句"

**核心要求：金句要反认知、颠覆常识、让人一惊**

**反认知金句的3种套路：**

| 套路 | 公式 | 示例 |
|-----|------|------|
| **套路1：A ≠ B** | 打破等式 | 认真 ≠ 有效<br>错题本 ≠ 抄题本<br>抄得多 ≠ 学得好 |
| **套路2：越X越Y** | 反直觉 | 错题本越做越薄，才是高手<br>想走得快，先学会"不做"<br>抄得越认真，提分越慢 |
| **套路3：最好的X是Y** | 颠覆定义 | 错题本最好的结局，是被扔掉<br>错题本最有价值的一页，是空白的那一页<br>最好的复习，是把错题"杀"掉 |

**反认知金句示例：**

| 类型 | 金句 | 为什么反认知 |
|-----|------|-------------|
| **颠覆型** | 抄得最认真的错题本，往往最没用 | 打破"认真=有效"的常识 |
| **颠覆型** | 错题本最好的结局，是被扔掉 | 大家都认为错题本要留着 |
| **颠覆型** | 90%的错题本，都是孩子的"安慰剂" | 揭示"假装努力"的真相 |
| **对立型** | 错题不是敌人，是最好用的"排雷工具" | 把"错误"重新定义为"工具" |
| **对立型** | 聪明的孩子，错题本越做越薄 | 大家认为错题本应该越来越厚 |
| **对立型** | 做错题本最快的方式，是不做错题本 | 从源头解决问题的思路 |
| **警示型** | 孩子花1小时抄错题，不如花10分钟想"为什么错" | 质疑"勤奋"的价值 |
| **警示型** | 把错题抄得再漂亮，不如把错因写得再丑 | 颠覆对"工整"的执念 |
| **警示型** | 错题本上写得最多的，应该是红笔，不是黑笔 | 强调"分析"重于"抄写" |
| **洞察型** | 错题本不是用来"复习"的，是用来"清零"的 | 重新定义错题本的作用 |
| **洞察型** | 真正会学习的孩子，在做错题之前就已经会了 | 强调预防的重要性 |
| **洞察型** | 错题本最有价值的一页，是空白的那一页 | 引发好奇，空白页用来做什么？ |
| **扎心型** | 你家孩子的错题本，是在感动谁？ | 质疑努力的对象 |
| **扎心型** | 错题本抄了3本，不如彻底弄懂1道题 | 质疑"量"的积累 |
| **扎心型** | 最无用的努力，就是把错题抄得整整齐齐 | 直击痛点 |

## 写作风格指南

### ✅ 要做到（叙事感与真人温度）

**句式段落层面：**
- 用完整的句子表达，一个观点用完整段落展开
- 段落之间有过渡和衔接，形成流畅的叙事
- 避免过度碎片化，不要每句话都是独立短句
- 可以有较长段落（4-6行），只要逻辑连贯、表达流畅

**叙事与情感层面：**
- 用"我"的第一人称视角，带个人观察和感受
- 加入具体场景、对话、细节描写
- 像和朋友聊天一样自然，有停顿、有思考、有情感
- 分享个人经历或观察到的案例，增加真实感
- 适当使用语气词和口语化表达（"其实"、"说实话"、"我观察到"）

**内容价值层面：**
- 多用场景化描述，让家长有代入感
- 给具体可操作的方法，不要只讲大道理
- 用真实案例增强说服力
- 每个观点要有展开，有解释，有例子

### ❌ 要避免

**句式层面：**
- 纯粹的短语堆砌（"焦虑。无助。绝望。"）
- 过度使用列表体（每篇都是1.2.3.4.5）
- 没有主语的句子频繁出现
- 每句话都是孤立短句，缺乏连贯性

**内容层面：**
- 纯粹贩卖焦虑，不给解决方案
- 学术化表达，像教科书
- 只有理论没有方法
- 标题党但内容不兑现
- 没有个人视角，像AI生成的内容
- 标题党但内容不兑现
