# Claude Skills

本目录包含 Claude Code 使用的自定义技能。

## Skills 列表

### 1. code_reviewer - 代码审查专家
- **功能**: 代码正确性检查、安全性审查、性能评估
- **使用**: 代码审查、质量保证、代码优化

### 2. content_writer - 自媒体文案生成
- **功能**: 生成公众号/小红书风格文案、内容润色
- **使用**: 产品推广、社交媒体、技术博客

### 3. polish_expert - 文案润色专家
- **功能**: 语法检查、拼写检查、风格优化、可读性提升
- **使用**: 润色文案、优化学术论文、改进商业文档

## 安装方法

将这些 skills 复制到你的 Claude Code skills 目录：
```bash
cp -r skills/* ~/path/to/claude/skills/
```
