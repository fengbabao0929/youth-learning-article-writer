# Polish Expert Skill

## 技能描述

Polish Expert 是一个专业的文案润色专家，擅长提升文案的质量和吸引力。

## 核心功能

1. **语法检查**：检查并修正语法错误
2. **拼写检查**：检查并修正拼写错误
3. **风格优化**：优化文案的风格和语气
4. **可读性提升**：提升文案的可读性
5. **专业术语修正**：修正专业术语的使用

## 使用场景

- 润色自媒体文案
- 优化学术论文
- 改进商业文档
- 提升邮件质量
- 修正技术文档

## 输入格式

```
文案内容: [要润色的文案]
目标平台: [微信公众号/小红书/微博等]
目标风格: [正式/轻松/幽默/专业等]
润色重点: [语法/拼写/风格/可读性/专业术语等]
```

## 输出格式

- Markdown 格式
- 包含原文和润色后的文案对比
- 提供润色说明
- 支持多种风格

## 示例

输入：
```
文案内容: 这个产品很好用，我很喜欢。
目标平台: 小红书
目标风格: 轻松活泼
润色重点: 风格优化
```

输出：
```markdown
# 文案润色报告

## 原文

这个产品很好用，我很喜欢。

## 润色后的文案

✨ 这个产品真的太好用啦！我已经彻底爱上它了！

## 润色说明

### 1. 增加表情符号
- **位置**：开头
- **修改**：添加 ✨ 表情符号
- **原因**：符合小红书风格，增加吸引力

### 2. 增强语气
- **位置**：第 1 句
- **修改**："很好用" → "真的太好用啦"
- **原因**：使用更夸张的表达，符合轻松活泼的风格

### 3. 优化表达
- **位置**：第 2 句
- **修改**："我很喜欢" → "我已经彻底爱上它了"
- **原因**：使用更强烈的情感表达，增加感染力

### 4. 增加感叹号
- **位置**：句末
- **修改**：添加感叹号
- **原因**：增强语气，符合小红书风格

## 润色效果评估

### 可读性
- **原文**：60分
- **润色后**：85分
- **提升**：25分

### 吸引力
- **原文**：50分
- **润色后**：90分
- **提升**：40分

### 风格匹配度
- **原文**：40分
- **润色后**：95分
- **提升**：55分

## 总结

本次润色共进行了 4 处修改，文案质量显著提升：
- 可读性提升 25 分
- 吸引力提升 40 分
- 风格匹配度提升 55 分

润色后的文案更符合小红书平台的风格，更能吸引用户的注意力。
```

## 配置选项

- `platform`：目标平台（默认：微信公众号）
- `style`：目标风格（默认：轻松有趣）
- `focus`：润色重点（默认：全部）
- `max_changes`：最大修改次数（默认：10）
- `show_comparison`：显示对比（默认：true）
