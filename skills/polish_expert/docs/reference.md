# Polish Expert 参考文档

## API 参考

### polishText(text, options)

润色文本并返回润色报告

**参数：**
- `text` (String)：要润色的文本内容
- `options` (Object)：配置选项
  - `platform` (String)：目标平台
  - `style` (String)：目标风格
  - `focus` (Array)：润色重点
  - `maxChanges` (Number)：最大修改次数
  - `showComparison` (Boolean)：显示对比

**返回值：**
- `Object`：润色报告对象
  - `original` (String)：原文
  - `polished` (String)：润色后的文本
  - `changes` (Array)：修改列表
  - `evaluation` (Object)：效果评估

**示例：**
```javascript
const text = '这个产品很好用，我很喜欢。';

const options = {
  platform: 'xiaohongshu',
  style: '轻松活泼',
  focus: ['style'],
  maxChanges: 10,
  showComparison: true
};

const report = await polishText(text, options);
console.log(report);
```

### checkGrammar(text)

检查文本的语法错误

**参数：**
- `text` (String)：要检查的文本

**返回值：**
- `Array`：语法错误列表

**示例：**
```javascript
const text = '我昨天去公园，看到一只漂亮的花。';

const errors = await checkGrammar(text);
console.log(errors);
```

### checkSpelling(text)

检查文本的拼写错误

**参数：**
- `text` (String)：要检查的文本

**返回值：**
- `Array`：拼写错误列表

**示例：**
```javascript
const text = '这个产品的质量很好，价格也很便易。';

const errors = await checkSpelling(text);
console.log(errors);
```

### optimizeStyle(text, style)

优化文本的风格

**参数：**
- `text` (String)：原始文本
- `style` (String)：目标风格

**返回值：**
- `String`：优化后的文本

**示例：**
```javascript
const text = '这个产品的性能很好，值得购买。';
const style = '轻松活泼';

const optimizedText = await optimizeStyle(text, style);
console.log(optimizedText);
```

## 润色类型参考

### 1. 语法检查

#### 常见语法错误
- 句子成分残缺
- 搭配不当
- 语序不当
- 句式杂糅
- 重复啰嗦

#### 修复方法
- 补充残缺的成分
- 调整搭配
- 调整语序
- 拆分杂糅的句式
- 删除重复的内容

### 2. 拼写检查

#### 常见拼写错误
- 同音字混淆
- 形近字混淆
- 错别字
- 漏字
- 多字

#### 修复方法
- 替换正确的字
- 添加遗漏的字
- 删除多余的字

### 3. 风格优化

- 使用规范的书面语
- 避免使用口语化表达
- 使用专业术语

- 使用搞笑的比喻
- 保持轻松愉快

#### 专业风格
- 使用行业术语
- 提供数据支持
- 引用权威来源
- 保持严谨准确

### 4. 可读性提升

## 常见问题

### Q: 润色后的文案会失去原文的意思吗？
A: 不会。润色的目的是提升文案的质量，而不是改变原文的意思。我们会尽量保持原文的核心信息。

### Q: 润色需要多长时间？
A: 润色的时间取决于文案的长度和复杂程度。一般来说，1000 字左右的文案需要 10-15 分钟。

### Q: 如何选择合适的风格？
A: 选择风格时需要考虑：
1. 目标平台
2. 目标受众
3. 写作目的
4. 内容类型

### Q: 润色后的文案一定更好吗？
A: 润色后的文案通常会比原文更好，但也可能不符合某些特定的需求。如果您对润色结果不满意，可以提供更多的反馈，我们会进行进一步的优化。

### Q: 可以润色其他语言的文案吗？
A: 目前主要支持中文文案的润色，未来会考虑增加其他语言的支持。
